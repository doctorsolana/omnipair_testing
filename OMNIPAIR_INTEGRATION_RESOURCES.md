# Omnipair Integration Resources

## Core Interfaces
- Program interface package (IDL + types): [@omnipair/program-interface](https://www.npmjs.com/package/@omnipair/program-interface)
- Historical API root: [https://api.indexer.omnipair.fi/](https://api.indexer.omnipair.fi/)

## Indexer Coverage
- Price and price history
- Pool stats, volume, and fees
- LP, borrow, and lending position data
- User-level swap, liquidity, and lending events
